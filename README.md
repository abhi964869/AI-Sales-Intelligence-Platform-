# 📊 AI Sales Intelligence Platform

An AI-powered Sales Analytics Dashboard built with **Python, Streamlit, Plotly, Pandas, and Google Gemini AI**. This application enables users to upload sales datasets in CSV or Excel format, explore interactive dashboards, and receive AI-generated business insights through natural language.

---

## 🚀 Live Demo

🌐 **Live Application:**
https://aisalesanalysisplatform.streamlit.app

💻 **GitHub Repository:**
https://github.com/abhi964869/AI-Sales-Intelligence-Platform

# 📌 Features

* 📂 Upload CSV and Excel datasets
* 🔄 Automatic column detection and mapping
* 📈 Interactive KPI dashboard
* 📊 Sales analysis by category and region
* 📉 Monthly sales trend visualization
* 🏆 Top-performing products and customers
* 🎯 Business insights powered by Google Gemini AI
* 💬 Ask AI questions about your uploaded sales data
* 📥 Export filtered data as CSV
* 📱 Responsive Streamlit interface

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/abhi964869/AI-Sales-Intelligence-Platform.git
```

### Navigate to Project

```bash
cd AI-Sales-Intelligence-Platform
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create Environment File

Create a `.env` file in the project root.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run the Application

```bash
streamlit run app.py
```
